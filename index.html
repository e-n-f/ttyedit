TTY editing patches

This web page contains links to the software described in my paper from the 1999 Usenix Annual Technical Conference, ``New Tricks for an Old Terminal Driver.'' The paper appears in the proceedings (Proceedings of the Freenix Track: 1999 Usenix Annual Technical Conference, Monterey, CA, June 6-11, 1999, pp. 219-221) or you can read it here in either PostScript or HTML format.

The patches are:

  • The kernel patch itself
  • A patch to stty so it can turn the new features on and off
  • A patch to mail that takes advantage of one of the new ioctls
  • The source for the daemon that maintains history lists

These are still evolving, so if you are thinking of including them in your operating system please contact me before you do.
